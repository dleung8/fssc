Menú de Ficheros

Nuevo:
Limpia el editor y comienza un nuevo fichero.
Abrir:
Abre un fichero de escenario existente.
Salvar:
Salva los datos actuales en el editor, usando el mismo nombre de fichero.
Salvar como:
Salva los datos actuales en el editor, usando un nombre distinto de fichero.
Exportar:
Exporta los datos actuales en el editor a Microsoft Flight Simulator en el formato adecuado usando las preferencias usadas.
Asistente para Exportar Escenario:
Exporta los datos actuales en el editor a Microsoft Flight Simulator con el asistente.
Lista de ficheros mas recientes:
Contiene los últimos cuatro ficheros abiertos.
Salir:
Acaba el programa.
