Objeto Área de Superficie

Una área de superficie puede definir una plataforma (como las usadas para las zona de helicópteros) o del tipo de terreno (la pista de aterrizaje, el césped, el agua). Para diseñar aeropuertos que no existen en el mundo de Flight Simulator 2002, debe usar un Área de Superficie con " el tipo Liso o Suave " para tener una zona donde poder aterrizar.
