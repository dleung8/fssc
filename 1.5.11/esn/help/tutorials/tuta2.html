Iniciación: Lección 2

Objetos

Entre la barra de herramientas y la barra de estado está el editor. Cuando usted empieza un nuevo paisaje, el editor contiene una cuadrícula que representa 1 kilómetro horizontalmente y verticalmente. Si usted hace click con el botón derecho en el editor, se presentará el menú del Editor que le permite agregar los objetos. (Alternativamente, usted puede apretar la tecla del Menú, o Shift+F10)

Como muchas aplicaciones de ordenador para diseño, el FS Scenery Creator usa el sistema de objetos. Cada elemento del paisaje (como un edificio o una pista de aterrizaje) es un objeto, y todos los objetos tienen una X e Y que le dice al programa dónde debe dibujar el objeto. Cada objeto también tiene un círculo alrededor dónde usted puede hacer clic para seleccionar el objeto. Cuando se seleccionó, este círculo y el objeto de muestran en azul luminoso.

Cuando usted pulsa otro objeto, el objeto anterior pierde la selección. Si usted quiere seleccionar los objetos de forma múltiple, mantenga presionado Ctrl mientras hace clic. Alternativamente, si usted hace clic dónde no hay ningún objeto y arrastra un rectángulo, se seleccionan todos los objetos que hay dentro del rectángulo dibujado por usted.
