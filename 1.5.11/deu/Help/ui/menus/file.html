Datei Menü

Neu:
Löscht die Arbeitsfläche und startet ein neues Projekt.
Öffnen:
Öffnet eine existierende Projektdatei.
Speichern:
Speichert die Daten der Arbeitsfläche mit Beibehalten des Dateinamens.
Speichern unter:
Speichert die Daten der Arbeitsfläche mit Vergabe eines neuen Dateinamens.
Importieren:
Importiert eine Microsoft Flight Simulator Scenery -(.bgl-)Datei.
Exportieren:
Exportiert die momentanen Daten der Arbeitsfläche in Microsoft Flight Simulator Format mit den zuletzt gewählten Voreinstellungen.
Export Scenery Wizard:
Exportiert die momentanen Daten der Arbeitsfläche in Microsoft Flight Simulator Format.
Dateiliste:
Enthält die letzten vier bearbeiteten Projektdateien.
Beenden:
Beendet das Programm.
