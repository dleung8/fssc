Polygon

Der Polygon definiert eine Fläche auf dem Terrain. Der Polygon kann mit einer Textur oder mit einer Farbe versehen werden. Optional können die Kanten mit leuchtenden Punkten ("taxilights") versehen werden, man kann mit Polygonen auch kleine Hügel erstellen, indem man einen Polygon mit einem anderen Polygon verbindet, welcher als Basis für das Plateau dient. FS Scenery Creator kann dann aus den beiden Polygonen die Aussenkontur des Hügels extrapolieren. Damit kann man allerdings beileibe keine harte, befahrbare Terrainoberfläche erstellen. Um dies zu tun, sollte man das Abflachgebiet verwenden.

Polygone werden durch mindestens drei oder fast beliebig viele Punkte definiert, die unabhängig voneinander verschoben werden können.
