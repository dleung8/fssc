Punkte

Der Punkt definiert den Umriss seines Objekts, nämlich eines HREF="reference\objects\polygon.html">Polygon, Taxiway, Taxiway Linie, Strasse, Fluss, Linie, und Abflachgebiets.

Neben den Positions-Eigenschaften hat der Punkt auch eine Typ-Eigenschaft. Die folgenden Typ-Eigenschaften können gewählt werden:

  • Sichtbar: Eine Linie (bzw. Taxiway etc.) wird zum nächsten Punkt weitergeführt.
  • Unsichtbar: Bis zum nächsten Punkt wird keine Linie gezeichnet, d.h. an dieser Stelle wird die sichtbare Linie unterbrochen.
  • punktiert: Bis zum nächsten Punkt werden helle Lichtpunkte sichtbar.
  • Taxiway: In regelmässigen Intervallen werden helle Lichtpunkte an dieser Kante des Polygons sichtbar.

Punkte können erzeugt werden, indem man auf eine bereits vorhandene Linie klickt, oder durch Drücken der "Einfg"-Taste. Punkte können gelöscht werden durch Drücken der "Entf"-Taste oder durch Benutzen der "Löschen"-Option im Bearbeiten Menü. Wenn die Zahl der Punkte unter das erlaubte Limit fällt, wird automatisch das gesamte Objekt gelöscht.

Die maximale Anzahl von Punkten, die ein Objekt haben kann, ist: 80 für den Combat Flight Simulator 1 und früher, 2000 für FS 2000 und neuer, und 30 für ein Abflachgebiet, welches die flatten switch Option benutzt.

Der Z-Parameter kann benutzt werden, um z.B. einen geneigten oder vertikalen Polygon zu erstellen. Allerdings sollte dies erfahrenen Anwendern vorbehalten bleiben.
