SCASM Code

SCASM Code schreibt die Rohdaten der Szenerie in einer Szenerie-Assembler-Sprache. Das Objekt wurde implementiert, um Standardobjekte mit einem Import-feature zu versehen und wird eigentlich im normalen Szeneriedesign nicht gebraucht. Die Dokumentation der SCASM-Assembler-Sprache ist im FSSC\Scasm-Unterverzeichnis zu finden.
