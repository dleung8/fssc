Linie

Wie der Name schon sagt, wird damit im Flugsimulator eine Linie dargestellt. Man kann damit unter anderem eine Mittellinie für einen Taxiway herstellen, oder Stromleitungen. Aber eine sehr leistungsfähige Eigenschaft ist es, dass Sie eine Linie als Perlschnur setzen können, auf der 3D-Objekte wie Gebäude oder Makros gesetzt werden, die dann in einem festgelegten Intervall wiederholt werden. Dazu braucht lediglich ein Objekt aus der Liste angewählt und das Wiederholungsintervall gesetzt zu werden. Vorsicht... man kann sich damit leicht die Darstellungsgeschwindigkeit des Flugsimulators zunichte machen, weil man damit leicht Hunderte von 3D-Objekten in die Landschaft setzt.

Linien werden durch zwei oder mehr Punkte definiert, die unabhängig voneinander verschoben werden können.
