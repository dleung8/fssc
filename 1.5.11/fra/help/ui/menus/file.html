Menu Fichier

Nouveau :
Vide le tampon de l'éditeur et commence un nouveau fichier (une nouvelle scène).
Ouvrir :
Ouvre un fichier de scène existant.
Enregistrer :
Enregistre les données en cours sans changer le nom du fichier.
Enregistrer sous :
Enregistre les données en cours sous un nouveau nom de fichier.
Exporter :
Exporte les données dans un format MS Flight Simulator.
Assistant d'exportation :
Aide à exporter les données dans un format MS Flight Simulator.
Liste des derniers fichiers :
Contient les noms des 4 fichiers les plus récemment utilisés.
Quitter :
Termine le programme.
