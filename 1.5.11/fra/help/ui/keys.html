Touches rapides

Les opérations suivantes peuvent être effectuées au clavier :

Nouveau :
Ctrl+N
Ouvrir :
Ctrl+O
Enregistrer :
Ctrl+S
Quitter :
Alt+F4
Couper :
Ctrl+X
Copier :
Ctrl+C
Coller :
Ctrl+V
Supprimer :
Suppr
Sélectionner un polygone :
Ctrl+A
Zoom +/Zoom variable :
Plus (+)
Zoom - :
Moins (-)
Aide :
F1
Rotation de 1 degré vers la droite :
Ctrl+R
Rotation de 1 degré vers la gauche :
Ctrl+L
Pour déplacer un objet d'un pixel (à l'échelle en cours) :
Ctrl+Flèche
Pour bloquer un objet :
Ctrl+K
Pour débloquer un objet :
Ctrl+U
Menu :
Alt ou F10
Menu contextuel (clic droit) :
Maj+F10 ou touche Menu
Propriétés par défaut (Tab 1) :
F4
Propriétés (Tab 2) :
F5 ou Maj+F4
Propriétés (Tab 3) :
F6
Propriétés (Tab 4) :
F7
Propriétés (Tab 5) :
F8
Insérer un point :
Inser
Insérer un symbole :
Ctrl+S (dans les champs acceptant une entrée texte)
