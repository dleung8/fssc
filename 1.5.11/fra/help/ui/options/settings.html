Préférences

Général

Mémorise l'état d'une fenêtre :
Conserve en mémoire la position de la fenêtre, même après l'arrêt et le redémarrage.
Liste de fichiers récents :
Affiche uniquement le nom court du fichier dans la liste des fichiers récents.
Afficher les propriétés de la scène avec un nouveau fichier :
Affiche les propriétés de la scène à toute ouverture de nouveau fichier.
Se rappeler les propriétés de l'objet précédent :
Mémorise les propriétés utilisées lors de la création d'un objet et les appliquera au prochain objet créé du même type.
Afficher les fractions de minutes :
Bascule entre les deux formats d'affichage de latitude/longitude.
Unité de mesure par défaut :
Sélectionne le système d'unités par défaut pour l'affichage de valeurs.
Orientation :
Sélectionne le système employé par défaut par le logiciel pour l'affichage des azimuts.
Menus à l'ancienne :
Affiche les menus de l'éditeur dans le style de FS Scenery Creator version 1.2 et précédentes.
Enregistrer les fichiers comprimés :
Retire toute information non essentielle lors de la sauvegarde de fichiers de FS Scenery Creator, pour économiser de l'espace disque. Noter que cette option n'a aucune influence sur la rapidité du programme.
Afficher l'Assistant d'Exportation à chaque fois :
Si cette case est décochée, la fonction Exporter utilisera les options de compilation de la scène précédente.
Utiliser les valeurs par défaut pour les macros :
Lors de la création d'une nouvelle macro, les champs de texte et les paramètres seront remplis par les valeurs par défaut. Décocher cette option pour conserver les valeurs lors de la sélection d'une nouvelle macro.

Version de Flight Simulator

Sélectionne la version de MS Flight Simulator ou Combat Floght Simulator pour laquelle une scène est créée.

Exportation

Éditer le fichier de configuration de FS :
Permet au logiciel de modifier le fichier Scenery.cfg de MS Flight Simulator, si celà est nécessaire.
Compresser les fichiers BGL :
Compresse automatiquement le fichier de scène créé.
Garder les fichiers temporaires :
Si cette case est cochée, les fichiers temporaires ne seront pas effacés.
Rappel de sauvegarde avant de compiler :
Vous demande si vous voulez enregistrer la scène avant de la compiler.

Apparence

Forme du pointeur :
Sélectionnez votre curseur préféré.
Cercles d'édition visibles :
Dessiner ces cercles autour du centre de chaque objet.
Marqueurs de points visibles :
Afficher un petit cercle repérant tous les points qui constituent un polygone, une voie de roulement, une route, un cours d'eau, une ligne ou une zone plane.
Dèplacer les points à la ligne la plus proche :
Placer les points sur la ligne la plus proche lors du déplacement d'un objet.
Objets texturés/colorés :
Remplir les polygones et objets de la couleur ou texture sélectionnée.
Lignes texturées/colorées :
Dessiner la voie de roulement, la route, le cours d'eau ou la ligne avec textures et couleurs.
Objets 3D texturés/colorés :
Dessiner le bâtiment et la macro avec textures et couleurs.
Afficher le compas :
Afficher le compas, nord magnétique en haut, dans le coin inférieur droit de l'éditeur.

Objets Visibles

Décocher cette option pour rendre l'objet sélectioné invisible dans l'éditeur.
