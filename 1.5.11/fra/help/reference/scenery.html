Propriétés de la scène

Ici, vous pouvez définir des paramètres concernant la scène entière.

Auteur :
Votre nom.
Droits d'auteur :
Entrer toute information nécessaire ici.
Nom :
Celui de la scène en cours de développement.
Code de langue :
Sélectionner la langue de votre version de MS Flight Simulator.
Région :
Sélectionner le nom de la région de la scène dans le menu déroulant.
Pays :
Entrer le nom du pays dans lequel la scène doit apparaître.
Etat :
Entrer si besoin le nom de l'état dans lequel la scène doit apparaître.
Ville :
Entrer le nom de ville sous lequel la scène doit figurer.
Identification ICAO :
Entrer l'appellation ICAO de l'aéroport.
Latitude/Longitude :
Définissent le centre de votre aéroport.
Rotation :
Entrez le nombre de degrés dont vous voulez décaler l'éditeur par rapport au nord vrai.
Horizontale/Verticale :
Entrez l'étendue de la scène.
Altitude :
Entrer l'altitude principale de la scène.
Déclinaison :
Entrer la déclinaison magnétique à l'endroit de votre scène. Dans la plupart des cas, celle-ci est insérée automatiquement par le programme en fonction de sa position géographique et ce champ est désactivé.
Exclusions :
Cocher les cases appropriées pour supprimer les objets de scènes de MS Flight Simulator si votre scène doit les remplacer.
Scène synthétique :
Pour MS Combat Flight Simulator 1 : sélectionne une texture comme arrière plan, et ses options. Si la scène par défaut dans MS Flight Simulator a déjà l'altitude et la texture correctes, cette sélection peut être "Aucune". Pour MS Flight Simulator 2000 et 2002, vous pouvez niveler une scène à une altitude donnée avec l'option "Transparent". Cependant, il est conseilé d'utiliser une zone plane à la place.
