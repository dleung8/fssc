SCASM Code Objet

L'objet Code SCASM est en fait une séquence de code en langage assembleur destiné à l'éditeur de scènes de FS Scenery Creator. Cet objet est assez rarement utilisé dans la conception de scènes par les non-professionnels. La documentation relative à SCASM se trouve dans le dossier FSSC\Scasm.
