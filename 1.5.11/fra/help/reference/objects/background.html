Objet Image d'arrière-plan

Un objet Image d'arrière-plan vous permet de placer une image sous la scène éditée, à la manière d'un calque, afin de mettre en place vos objets avec plus de précision. Si la case "visible" est cochée, l'image sera alors également affichée dans le paysage de Flight Simulator.

Pour une précision totale, il vous faudra indiquer soit l'échelle de l'image en mètres par pixel (ou toute autre unité de distance compatible), soit les coorddonées géographiques de deux coins opposés de l'image et son orientation.

Note : Si vous choisissez de rendre votre image d'arrière-plan visible, FS Scenery Creator créera un bitmap dans le dossier de sortie des textures dans un format compatible avec MS Flight Simulator. Si vous redistribuez votre scène à d'autres amateurs, n'oubliez pas d'inclure ce fichier image !
