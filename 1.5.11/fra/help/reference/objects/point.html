Objet Point

Le Point définit la forme d'un objet parent qui peut être un polygone, une voie de roulement, une route, un cours d'eau, une ligne ou une zone plane.

Outre sa position, le point possède aussi un type. Les choix suivants ne sont pas tous disponibles pour tous les types d'objets :
  • Visible : Une ligne (ou voie de roulement, etc.) s'affiche au point suivant.
  • Invisible : Aucune connexion ne sera visible entre ce point et le suivant, et aucune ligne ne sera affichée.
  • Pointillé : Des tirets/pointilléss seront affichés jusqu'au point suivant.
  • Voie de roulement : Une ligne pointillée est dessiné le long du bord du polygone.

    Les points sont créés en cliquant sur une ligne de l'objet parent ou par pression sur la touche Inser. Ils sont supprimés par la touche Suppr, ou par l'option suppression du menu Editer.

    En-desous d'un nombre de points minimal, l'objet parent sera automatiquement supprimé. Le nombre maximal de points qui peuvent être créé dans un objet parent est de : 80 pour Combat Flight Simulator 1, 2000 pour Flight Simulator 2000 et versions ultérieure, et 30 pour une zone plane définie par une balise Aplanir.

    Le facteur Z peut être utilisé pour créer des effets divers, comme par exemple des polygones inclinés. Toutefois, ce facteur ne devrait être manipulé que par les utilisateurs expérimentés.
