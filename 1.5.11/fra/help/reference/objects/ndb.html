Objet NDB

Un NDB (Non-Directional radio Beacon) est un autre type de radiobalise employée dans la navigation aérienne. Pour en savoir plus, consulter le manuel de Flight Simulator.

Si la case "Entrée" est cochée, les informations du NDB seront insérées dans la base de données du répertoire des aéroports et aménagements, également utilisé par la carte du GPS.
