Texture

Les Textures remplissent la plus grande partie des paysages de MS Flight Simulator. Les objets sont presque tous habillés d'une texture. Dans FS Scenery Creator, vous disposez d'une liste de textures pour des objets tels que les bâtiments et les pistes et vous utiliserez les textures de votre système pour habiller par exemple les polygones.

A partir de MS Flight Simulator 2000, des textures différentes peuvent être choisies suivant les heures de la journée et les saisons. Ces textures doivent être nommée d'après une convention qui les rendent détectables par Flight Simulator. La liste ci-dessous présente cette convention :

  • Normal : base.bmp
  • Nuit : base_LM.bmp
  • Printemps : base_SP.bmp
  • Automne : base_FA.bmp
  • Hiver : base_WI.bmp
  • Hiver rigoureux : base_HW.bmp

Malheureusement, peu de bitmaps courants peuvent être utilisés directement par Flight Simulator, car ils doivent être auparavant convertis par un utilitaire graphique tel que BMP2000, DxtBMP ou ImageTool de Microsoft.
