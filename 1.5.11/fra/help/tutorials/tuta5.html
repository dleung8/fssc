Premiers pas avec FS Scenery Creator : Leçon 5

Pour conclure...

Voici achevée pour vous la découverte de base de FS Scenery Creator.

Si vous avez besoin d'aide pour un champ de texte particulier dans une page de propriétés, cliquer sur en haut à droite de n'importe quelle fenêtre, puis choisissez le sujet concerné. Ceci affichera un petit message d'explication sur le sujet recherché.

Si vous considérez que quelque chose manque à ce premier tutoriel, vous pouvez contacter l'auteur par courrier électronique à fsschelp@yahoo.com.
