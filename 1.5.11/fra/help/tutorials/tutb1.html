Les Bases : Leçon 1

Quelques notes:

Les différentes versions de Flight Simulator présentent quelques spécificités en ce qui concerne le développement de scènes. Prenez garde à noter ces différences dans les tutoriels correspondant à chacune de ces versions (FS95, FS98, CFS1, FS2000, CFS2, FS2002).

Prenez le temps de lire pour commencer celui intitulé Premiers pas avec FS Scenery Creator.

Ce premier tutoriel est important. Son titre indique bien quel est son but : vous guider dans vos premiers pas dans la découverte de la conception de scène, et de vous aider - pour commencer - à développer une scène miniùale. Bonne chance !

Le présent tutoriel présuppose que vous utilisiez le jeu de couleurs par défaut de FS Scenery Creator. Vous devriez aussi vous assurer que vous avez coché la bonne case concernant la version de Flight Simulator pour laquelle vous travaillez, dans la fenêtre des Préférences, car certaines caractéristiques diffèrent entre versions.

Démarrage

Si FS Scenery Creator n'est pas encore démarré, faites-le maintenant : Menu Démarrer, Programmes, Leung Software, FS Scenery Creator.
